BMI Calculator

This project was one of my first apps made. I built it as a practice project while doing a udemy course on App Development.

The app is a simple BMI calculator that takes in a user's height and weight and calculates their BMI. The app then displays the user's BMI and a message based on their BMI.

The app was built using Dart with the Flutter framework. The app was built for Android devices and was tested on a Pixel 3a.
